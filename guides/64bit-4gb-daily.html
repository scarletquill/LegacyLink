CachyOS + Hyprland

Push your legacy hardware to its maximum potential. Combining CachyOS's CPU-optimized packages with the fluid animations of Hyprland delivers an ultra-modern, blazing-fast Wayland experience.

Why This Stack

  • BORE Scheduler: CachyOS ships with the Burst-Oriented Response Enhancer (BORE) scheduler by default, which yields the best interactive desktop responsiveness currently available on Linux.
  • ALHP Repo (x86-64-v3): Arch Linux x86-64-v3 packages are compiled to use modern CPU instructions (like AVX2). This squeezes significantly more performance out of your 64-bit processor than standard generic binaries.
  • Hyprland: The most feature-rich dynamic tiling Wayland compositor. It provides incredibly smooth animations and blur effects while keeping base RAM usage remarkably low compared to KDE or GNOME.
  • XFS Filesystem: A high-performance journaling file system that handles large media and game files exceptionally well on both mechanical drives and early SSDs.
  • Zen Browser: A heavily optimized, privacy-respecting Firefox fork designed for speed, ensuring your web experience keeps pace with the OS.

Prerequisites

  • A 64-bit PC with 4GB+ of RAM (Must support the x86-64-v3 instruction set. CPUs from ~2013 onwards usually do).
  • A blank USB Flash Drive (8GB or larger).
  • A secondary computer to flash the ISO.
  • A reliable internet connection.

Step 1: Download & Flash

Download the latest CachyOS Desktop ISO from their official site. It comes pre-loaded with the necessary performance patches.

Download CachyOS ISO →

Flashing via Linux Terminal

bash
sudo dd if=cachyos-desktop-linux-x86_64.iso of=/dev/sdX bs=4M status=progress

(Replace /dev/sdX with your actual USB drive identifier. Do not append partition numbers).

Step 2: Installation Walkthrough

  1. Boot from the USB drive. The live environment will automatically start.
  2. Launch the CachyOS Installer (Calamares) from the welcome screen.
  3. When prompted for a Desktop Environment, select Hyprland.
  4. Partitioning: Choose "Erase disk". In the advanced settings or manual partitioner, ensure your root partition is formatted as XFS.
  5. Configure your user account, timezone, and hostname.
  6. Let the installer run (it will pull optimized packages from the internet). Reboot when finished.

Step 3: Post-Install Optimization

While CachyOS handles much of the optimization out of the box, we will verify the scheduler and explicitly enable the ALHP repositories for maximum performance.

1. Enabling ALHP Repositories

ALHP provides Arch packages compiled with x86-64-v3 optimizations. Edit your pacman.conf to prioritize these.

bash
sudo nano /etc/pacman.conf

Uncomment or add the ALHP repository lines above the standard core/extra repos:

pacman.conf
[core-x86-64-v3] Include = /etc/pacman.d/alhp-mirrorlist [extra-x86-64-v3] Include = /etc/pacman.d/alhp-mirrorlist

Sync the new databases and upgrade: sudo pacman -Syu

2. ZRAM Verification

CachyOS sets up a high-performance ZRAM block device automatically. No manual setup is needed, but you should verify it's active.

Step 4: Install the App Stack

Fetch the essential GUI applications that pair perfectly with Hyprland.

bash
sudo pacman -S hyprland kitty dolphin zen-browser

Hyprland Config Basics

Configure your keybindings and set Kitty as the default terminal. Open the configuration file:

bash
nano ~/.config/hypr/hyprland.conf

Look for the $terminal variable and set it, along with keybindings for your apps:

hyprland.conf
$terminal = kitty $fileManager = dolphin $browser = zen-browser bind = $mainMod, Return, exec, $terminal bind = $mainMod, B, exec, $browser bind = $mainMod, E, exec, $fileManager

Save the file. Hyprland hot-reloads configurations instantly.

Step 5: Verification

Ensure the kernel scheduler, Wayland compositor, and memory compression are running at peak efficiency.

Verify BORE Scheduler

bash
cat /sys/kernel/debug/sched/features | grep BORE

Output should explicitly highlight the BORE feature flag as active.

Verify Hyprland Version

bash
hyprctl version

Verify ZRAM Status

bash
zramctl free -h

You will see your physical RAM usage, alongside the ZRAM device managing compressed swap data.

Troubleshooting

Hyprland boots to a black screen

If you are using an older NVIDIA GPU, Wayland compositors like Hyprland require specific environment variables to function correctly. Open ~/.config/hypr/hyprland.conf from a TTY (Ctrl+Alt+F3) and add env = WLR_NO_HARDWARE_CURSORS,1 and env = LIBVA_DRIVER_NAME,nvidia.

Pacman errors out on ALHP keys

If pacman complains about unknown or invalid PGP keys when syncing the ALHP repository, you need to install the ALHP keyring first. Run sudo pacman -S alhp-keyring before attempting your full system upgrade.

Zen Browser lacks hardware acceleration

To ensure Zen is offloading video decoding to your GPU, go to about:config, search for media.ffmpeg.vaapi.enabled, and set it to true. You may also need to install the appropriate libva driver for your specific graphics card.
