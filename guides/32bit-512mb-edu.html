Puppy Linux (Fossapup32)

An exceptionally responsive, portable educational workstation that operates entirely in RAM via SFS layering, transforming 512MB systems into capable study machines.

Why This Stack

  • SFS Layering: Puppy Linux copies its core file systems (SFS files) directly into RAM upon boot. This delivers SSD-like responsiveness, even on severely bottlenecked 512MB IDE/SATA setups.
  • aufs Persistence: By creating a "pupsave" file, you gain persistent storage on a USB drive without requiring a full hard drive installation, reducing disk wear completely.
  • Gnumeric: A specialized, ultra-lightweight spreadsheet processor that consumes a fraction of the RAM required by LibreOffice Calc, perfect for math and data assignments.
  • ePDFView: An extremely lean PDF reader for accessing textbooks, worksheets, and lecture notes without crashing limited memory systems.
  • Falkon-lite: Provides capable, basic web browsing for educational research, optimized for low memory overhead compared to modern Chrome or Firefox.

Prerequisites

  • A 32-bit (x86) legacy PC with at least 512MB RAM.
  • A blank USB Flash Drive (2GB or larger recommended for pupsave space).
  • A secondary computer to download and flash the ISO.
  • Internet access via Ethernet or a supported Wi-Fi adapter.

Step 1: Download & Flash

Download the 32-bit version of Puppy Linux (such as Fossapup32 or Bionicpup32) from the official repository.

Download Puppy Linux 32-bit ISO →

Flashing via Linux Terminal

bash
sudo dd if=fossapup32.iso of=/dev/sdX bs=4M status=progress

(Replace /dev/sdX with your actual USB drive identifier, e.g., /dev/sdb. Do not append partition numbers).

Step 2: Installation Walkthrough

  1. Insert the USB into the legacy PC and boot from it (press F2, F12, or DEL to access boot options).
  2. Puppy will automatically load its core SFS files into your RAM. This process may take a minute or two on older USB 2.0 ports.
  3. Once the desktop appears, you will be greeted by the Quick Setup window.
  4. Configure your Country, Timezone, and Keyboard layout. Check the box to synchronize time with the internet if connected.
  5. Click OK to restart the X server (graphical interface) and apply the changes.
  6. Note: Puppy Linux is fully operational right now. A formal hard drive installation is not required!

Step 3: Post-Install Optimization

1. Creating the pupsave (Persistence)

To ensure your files, apps, and settings are saved, you must create a pupsave file. This happens during your very first shutdown or reboot.

  1. Click Menu -> Leave -> Reboot.
  2. A prompt will appear asking if you want to save your session. Select Save.
  3. Choose your USB drive as the save location.
  4. Choose ext4 as the file system for the save file.
  5. Set the size (512MB to 1GB is usually sufficient for schoolwork). Puppy will generate the file and reboot.

2. Enable ZRAM

On 512MB systems, enabling ZRAM provides essential compressed swap space inside the RAM, preventing system lockups during heavier web browsing.

  1. Navigate to Menu -> System -> Puppy Setup.
  2. Select the Swap/ZRAM configuration tool.
  3. Enable ZRAM and allocate 256MB. Apply the settings.

Step 4: Install the App Stack

Puppy Linux uses the Puppy Package Manager (PPM), but you can also use terminal commands to quickly fetch the required educational tools.

Open the console (terminal) and run the following command:

bash
puppy-pkg install gnumeric epdfview falkon

Note: Depending on your specific Puppy version, the exact package names might vary slightly. You can always use the graphical PPM (Menu -> Setup -> Puppy Package Manager) to search for Gnumeric and ePDFView.

Step 5: Verification

Ensure your system is running lean and your applications are properly layered.

Check RAM Usage & ZRAM

bash
free -h

You should see your ZRAM partition listed under Swap, and your primary memory usage should be extraordinarily low.

Verify Installed Educational Apps

bash
which gnumeric epdfview

This will return the paths to the installed binaries, confirming successful deployment.

Troubleshooting

Save File (pupsave) is filling up rapidly

If you download large PDFs or media, save them *outside* the pupsave space directly to the root of the USB drive (usually accessible in /mnt/home). Keep the pupsave strictly for software configurations and lightweight documents.

No Wi-Fi Connection

Click the network icon in the bottom right tray and select Frisbee or Simple Network Setup (SNS). These built-in tools are excellent at detecting older 32-bit Broadcom and Atheros Wi-Fi cards natively.

Falkon-lite crashes on heavy modern sites

Modern web browsing on 512MB RAM is inherently difficult due to JavaScript bloat. For pure research, consider disabling JavaScript in Falkon's settings, or using a text-based browser like dillo for text-heavy Wikipedia searches.
