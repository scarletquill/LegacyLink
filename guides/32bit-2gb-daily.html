AntiX Linux

Reclaim your desktop's performance. By pairing the ultra-light runit init system with aggressive RAM compression, AntiX turns 2GB of memory into a highly responsive daily workflow.

Why This Stack

  • runit Init System: Systemd can be heavy on older hardware. AntiX ships with runit, drastically reducing boot times and background service overhead.
  • ZRAM at 50%: Compressing half of your physical memory essentially provides ~3GB of usable space, allowing modern web browsing without triggering agonizing hard drive swap thrashing.
  • BFQ Scheduler: The Budget Fair Queuing scheduler prioritizes interactive tasks (like moving windows and opening menus), making mechanical HDDs feel significantly snappier.
  • IceWM: A blazing fast, keyboard-friendly window manager that uses negligible RAM while offering a familiar, taskbar-driven interface.
  • Pale Moon: A modern, independently developed browser that natively maintains robust 32-bit architecture support, avoiding the bloat of contemporary Chrome/Firefox.

Prerequisites

  • A 32-bit (x86) legacy PC with 2GB of RAM.
  • A blank USB Flash Drive (4GB or larger).
  • A secondary computer for flashing the ISO image.
  • A standard Ethernet connection (highly recommended to simplify initial software fetching).

Step 1: Download & Flash

Download the antiX-full (32-bit runit) ISO from the official AntiX Linux SourceForge repository. Ensure you explicitly select the runit edition over the sysVinit one.

Download AntiX runit 32-bit ISO →

Flashing via Linux Terminal

bash
sudo dd if=antiX-23-runit_386-full.iso of=/dev/sdX bs=4M status=progress

(Replace /dev/sdX with your actual USB drive identifier, e.g., /dev/sdb. Do not append partition numbers).

Step 2: Installation Walkthrough

  1. Boot the target machine from the USB drive.
  2. At the bootloader menu, select the default AntiX boot option.
  3. Once the live desktop loads, click the Install icon located on the desktop.
  4. Follow the graphical installer. When prompted for disk partitioning, select Custom Install or auto-use the whole disk. (ext4 is recommended for general 2GB systems).
  5. Set your username, root password, and timezone.
  6. Disable unnecessary services on the "Services" screen (e.g., Bluetooth if you don't have it) to save RAM.
  7. Reboot the system, remove the USB drive, and log in to your fresh AntiX installation.

Step 3: Post-Install Optimization

Open the RoxTerm terminal and apply the core structural optimizations.

1. Configure 1GB ZRAM (50% compression)

Run these commands to immediately initialize ZRAM.

bash
sudo modprobe zram sudo zramctl /dev/zram0 --size 1G sudo mkswap /dev/zram0 sudo swapon /dev/zram0 -p 10

To make this persist across reboots in antiX (runit), add the above commands to your /etc/rc.local file just before the exit 0 line.

2. Apply the BFQ I/O Scheduler

Inject BFQ into your hard drive's queue to prioritize UI responsiveness.

bash
echo bfq | sudo tee /sys/block/sda/queue/scheduler

(Replace sda with your drive letter if different). To make this permanent, edit /etc/default/grub and add elevator=bfq to the GRUB_CMDLINE_LINUX_DEFAULT line, then run sudo update-grub.

Step 4: Install the App Stack

AntiX utilizes Debian's APT package manager. Execute the following to retrieve your daily driver suite:

bash
sudo apt update sudo apt install icewm palemoon-bin thunar
  • IceWM: Though AntiX comes with a few WMs, IceWM is pre-configured perfectly here for traditional desktop users.
  • Pale Moon (palemoon-bin): A fork of Firefox focused on efficiency. It strips out telemetry and heavy UI elements while supporting legacy extensions.
  • Thunar: XFCE's default file manager; vastly superior for daily file operations compared to Rox-Filer, while still maintaining a very light memory footprint.

Step 5: Verification

Confirm that your optimizations have been applied successfully.

Verify ZRAM Activation

bash
zramctl

You should see /dev/zram0 listed with a SIZE of 1G.

Verify Scheduler

bash
cat /sys/block/sda/queue/scheduler

The output should highlight BFQ like this: noop deadline [bfq].

Troubleshooting

No Wi-Fi Connection Available

AntiX uses connman instead of NetworkManager. Click the network icon in your system tray, or open a terminal and run sudo connman-ui-gtk or ceni to configure your wireless adapters manually.

Screen Tearing while Scrolling

IceWM does not inherently composite windows. If you experience screen tearing, install a lightweight compositor. Run sudo apt install picom and add picom & to your ~/.icewm/startup file.

Pale Moon feels sluggish on heavy sites

While Pale Moon is highly optimized, modern JavaScript is inherently demanding. Go to Preferences -> Advanced -> Performance and ensure Hardware Acceleration is checked. Consider installing the uBlock Origin Legacy extension to block heavy background scripts.
